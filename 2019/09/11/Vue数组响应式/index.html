九月 11, 2019
  • Vue
  • 数组

可以实现数组响应式的方法

  • arr.push(); //向数组中添加元素
  • arr.pop(); //删除数组中最后一个元素
  • arr.shift(); //删除数组中第一个元素
  • arr.unshift(); //向数组中最前面添加一个元素
  • arr.splice(); //删除数组中元素/向数组中插入元素/修改数组中元素
  • arr.sort(); //正序排列数组
  • arr.reverse(); //倒序排列数组

不能实现数组响应式的方法

  • arr[0] = ‘a’； //通过索引值修改数组中元素
