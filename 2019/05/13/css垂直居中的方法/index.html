CSS垂直居中的7种方法

CSS垂直居中的7种方法

五月 13, 2019
  • css
  • 垂直居中

在编辑一个界面时，我们通常都会用到水平居中和垂直居中来设计，而水平居中很好处理，不外乎就来设定margin:0 auto，或者是text-align:center;就可以轻松解决掉水平居中的问题。但一直以来最麻烦的对齐问题，都是“垂直居中”这个讨人厌的设定，下面就是7种纯css垂直居中的方式。

七种垂直居中的方法

  • 设定行高 line-height
  • 添加伪元素
  • calc()动态计算
  • 使用表格或假装表格
  • transform
  • 绝对定位
  • 使用FlexBox

设定行高

设定行高是垂直居中最简单的方式，适用于“单行”的“行内元素”（inline、inline-block），例如单行的标题，或是已经设为inline-block属性的div，若将line-height设成和高度一样的数值，则内容的行内元素就会被垂直居中，因为是行高，所以会在行内元素的上下都加上行高的1/2，所以就垂直居中了！不过由此就可以看出，为什么必须要单行的行内元素，因为如果多行，第二行与第一行的间距会变超大，就不是我们所期望的效果了。CSS示例：
