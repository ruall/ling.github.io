定义函数

  • 声明（定义）函数必须用function关键字
  • 函数参数可有可无，多个参数用,隔开
  • 每个函数由名称和类型组成
  • 函数的返回值可有可无，没有时返回viod
  • 大括号里的内容就是函数体

1、可选参数（用？来标识）

2、默认参数（直接用=赋值）

3、不确定参数 （用...标识）

声明数组（在TypeScript中指定数据类型的数组只能存储同一类型的数组元素）

元祖（一种特殊的数组，元祖类型可以表示一个已知元素数量和类型的数组，各元素的类型不必相同）

字符串类型

  • 基本类型的字符串 let str:string = 'a'
  • 引用类型的字符串 let str:String = new String('a')

查找字符串 indexOf

截取字符串 substring

str.substring(startIndex,[endIndex])

  • startIndex 开始下标
  • endIndex 结束下标，endIndex的参数是可选的，如果不填表示截取到字符串末尾

替换字符串 replace

str.replace(subStr,newstr);

  • str 表示被替换的字符串
  • newstr 表示需要替换成的字符串
