number类型1加上""可以强转为string类型 （1+""=>string）

对象（Object）类型

在JS中，除了原始类型，其他都是对象类型，对象类型和原始类型的不同之处在于原始类型存的是值，而对象类型存的是路径（指向值的指针）

typeof 和 instanceof

typeof 对于原始类型，除了 ‘null’ 都可以显示正确类型

typeof 对于对象类型，除了函数其他都会显示 ‘object’ ，typeof并不能准确判断变量是什么类型
