原始类型存储的都是值，是没有方法可以调用的

typeof 和 instanceof

typeof 对于原始类型，除了 ‘null’ 都可以显示正确类型

typeof 1 =>number
typeof ‘1’ => string
typeof undefined => undefined
typeof true => boolean
typeof Symbol() => symbol

typeof 对于对象类型，除了函数其他都会显示 ‘object’ ，typeof并不能准确判断变量是什么类型

typeof [] => ‘object’
typeof {} => ‘object’
typeof console.log => ‘function’
